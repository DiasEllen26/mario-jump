# Mario Jump 🍄🎮

Bem-vindo ao repositório do jogo Mario Jump! Neste projeto, desenvolvi um divertido jogo baseado no famoso personagem Mario, onde ele deve pular nos canos para alcançar alturas incríveis. Sinta-se à vontade para jogar e aproveitar a nostalgia! 🕹️🌟

## Demo Online 🌐

Você pode jogar o Mario Jump online acessando o seguinte link: [Demo do Jogo](https://diasellen26.github.io/mario-jump/)

## Como Jogar 🚀

1. Acesse a [Demo do Jogo](https://diasellen26.github.io/mario-jump/) usando o link fornecido acima.

2. Pressione a tecla de espaço para fazer o Mario pular nos canos.

3. Evite colidir com os canos. Quanto mais alto você chegar, mais pontos acumulará!

4. Seu objetivo é chegar o mais longe possível no jogo.

## Tecnologias Utilizadas 🛠️

- HTML5: Para a estrutura básica do jogo.
- CSS3: Para estilização e layout responsivo.
- JavaScript: Para a lógica do jogo e interações com o usuário.

## Como Executar Localmente 🏃‍♂️

Para executar o jogo em sua máquina local, siga os passos abaixo:

1. Clone o repositório para o diretório desejado em seu computador:

```bash
git clone https://github.com/DiasEllen26/mario-jump.git
```

2. Navegue até a pasta do projeto:

```bash
cd mario-jump
```

3. Abra o arquivo `index.html` no seu navegador preferido.

4. Jogue e divirta-se!
